# The Internet Test Automation (JavaScript)

This repository contains automated test scenarios for  
https://the-internet.herokuapp.com

The project is created to practice real-world web test automation using JavaScript.

## Tech Stack
- JavaScript
- Mocha
- Chai
- Page Object Model

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm test`
